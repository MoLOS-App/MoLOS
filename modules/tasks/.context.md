# Tasks Module - AI Context

## Purpose
Task management module for MoLOS providing full CRUD operations, UI components, and server-side logic for managing tasks.

## Key Files
- `src/index.ts` - Main entry point
- `src/config.ts` - Module configuration and registration
- `src/components/` - Svelte components for task UI
- `src/models/` - Data models and types
- `src/routes/` - API routes for task operations
- `src/server/` - Server-side logic and repositories
- `src/stores/` - Svelte stores for state management

## Import Patterns
```typescript
// Import components
import { TaskList, TaskForm } from '@molos-modules/tasks';

// Import stores
import { taskStore } from '@molos-modules/tasks';
```

## Module Structure
1. **Components** - UI components for displaying and editing tasks
2. **Models** - Type definitions and data models
3. **Routes** - REST API endpoints
4. **Server** - Business logic and database operations
5. **Stores** - Client-side state management

## Common Tasks
1. Adding new task fields - Update models, schema, and migrations
2. Adding new UI components - Create in src/components/
3. Adding new API endpoints - Create in src/routes/

## Dependencies
- `@molos/core` - Core types
- `@molos/database` - Database access
- `@molos/ui` - UI components
- `svelte` - UI framework
