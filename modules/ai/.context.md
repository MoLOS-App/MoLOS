# AI Module - AI Context

## Purpose
AI module for MoLOS providing AI-related functionality and integrations.

## Key Files
- `src/index.ts` - Main entry point for the module
- `src/config.ts` - Module configuration

## Module Structure
This module follows the MoLOS module pattern:
- Config exports for module registration
- Integration with the main application via module system

## Import Patterns
```typescript
// Import from module
import { aiConfig } from '@molos-modules/ai';
```

## Common Tasks
1. Extending AI capabilities - Add new functions to src/
2. Configuration changes - Modify src/config.ts
3. Integration with external AI services - Add API clients

## Dependencies
- `@molos/core` - Core types and utilities
- `@molos/database` - Database access
