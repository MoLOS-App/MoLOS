# @molos/ui - AI Context

## Purpose

UI component library for MoLOS built with Svelte 5, shadcn-svelte, and Tailwind CSS. Provides reusable components for the application and modules.

## Key Files

- `src/index.ts` - Main entry point
- `src/components/ui/` - shadcn-svelte components (Button, Card, Dialog, etc.)
- `src/components/shared/` - Shared application components
- `src/lib/utils.ts` - Utility functions including `cn()` for class merging
- `src/styles/` - Global styles and Tailwind config

## Import Patterns

```typescript
// Import UI components
import { Button, Card, Dialog } from '@molos/ui';

// Import shared components
import { ComponentName } from '@molos/ui';

// Import utilities
import { cn } from '@molos/ui';
```

## Component Categories

1. **UI Components** (`src/components/ui/`) - shadcn-svelte primitives
   - Button, Card, Dialog, Dropdown, Form, Input, etc.
2. **Shared Components** (`src/components/shared/`) - Application-specific reusable components

## Common Tasks

1. Adding new shadcn components - Use `npx shadcn-svelte@latest add <component>`
2. Creating shared components - Add to `src/components/shared/`
3. Styling - Use Tailwind classes with `cn()` utility

## Dependencies

- `svelte` - UI framework
- `tailwindcss` - CSS framework
- `clsx`, `tailwind-merge` - For `cn()` utility
- shadcn-svelte components
