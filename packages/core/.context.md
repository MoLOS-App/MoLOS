# @molos/core - AI Context

## Purpose
Core utilities and shared types for the MoLOS application. This package provides foundational types and utility functions used across all other packages and modules.

## Key Files
- `src/index.ts` - Main entry point, exports all utilities and types
- `src/types/index.ts` - Exports all type definitions
- `src/types/common.ts` - Common shared types
- `src/types/module.ts` - Module-related types
- `src/types/ai.ts` - AI-related types
- `src/types/mcp.ts` - MCP (Model Context Protocol) types
- `src/types/search.ts` - Search-related types
- `src/utils/` - Utility functions

## Import Patterns
```typescript
// Import types
import type { Module, AIConfig, MCPConfig } from '@molos/core';

// Import utilities
import { someUtility } from '@molos/core';
```

## Common Tasks
1. Adding new shared types - Add to appropriate file in `src/types/`
2. Adding new utilities - Add to `src/utils/` and export from index
3. Updating module types - Modify `src/types/module.ts`

## Dependencies
- Minimal dependencies - designed to be lightweight
- No framework-specific dependencies
